Bootcamp 2020: 2D Ping-Pong Game

My first programming project was made during one of the practicals before I found out that we are having a project the next week. When completed the tasks I've tried to make this little bouncing ball game we all know from our childhood (at least I remember..).
I've created a circle that moves with a randomized velocity. At start the range is from 5 to 6 but it speeds up with every picked point that appears in random places on the screen. This project demonstrates an understanding of proper if statements and how to set the mathematical conditions.
